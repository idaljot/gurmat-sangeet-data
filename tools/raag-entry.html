How to use this tool
  1. Pick a raag from the list on the left — sub-raags are indented under their parent (e.g. 3.1 under 3).
  2. Click the crop image(s) to zoom in — read them closely, especially for komal/teevra marks.
  3. Fill in the fields. Amber-highlighted fields are unverified OCR guesses — retype them once you've checked the image, especially Aroh/Avroh/Mukh Ang, which must be canonical SARGAM (Roman letters, see SARGAM.md), not the book's own Gurmukhi notation.
  4. Your typing autosaves as you go (to this browser only). Click Mark as reviewed once you're confident an entry is done.
  5. When you're done for the session, use Download backup in the sidebar so your work survives a browser/cache clear — you can hand that file to Daljot or reload it later with Restore backup.
